WeatherModule

This is the class storing every functions used to handle the weather

Functions​

SetClouds​

</>
WeatherModule.SetClouds(
CloudsCover: Float,--

Changes how cloudy the sky is (between 0 and 1)

CloudsDensity: Float,--

Changes how strong or thick the clouds are (between 0 and 1)

CloudsColor: Color3--

Changes the color of the clouds

) → ()

Used to change clouds (cover, density and color).

local WeatherModule = [Location Of The Module]

WeatherModule.SetClouds(0.5, 0.7, Color3.fromRGB(255,255,255))
info
  • 'CloudsCover' and 'CloudsDensity' can have values above 1, but it will not be different than setting it to 1.
  • Right now it takes 5s for the game to complete the modification, to change that just change the value in 'TweenInfo.new(5)'

SetWind​

</>
WeatherModule.SetWind(
WindPower: Float,--

Changes how strong the wind is.

WindSpeed: Float--

Changes how rapidly objects hit by the wind move

) → ()

Used to change the speed and power of the wind by changing the attributes of the 'WindShake' module.

caution

Changing the wind way too fast may result in the game lagging and objects supposedly hit by the 'WindShake' module to not move as they do not have enough time to adapt to the new settings 

local WeatherModule = [Location Of The Module]

WeatherModule.SetWind(0.5, 2)
